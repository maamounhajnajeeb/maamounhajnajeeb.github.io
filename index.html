Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', Determine if the input string is valid.

An input string is valid if:
  • Open brackets must be closed by the same type of brackets.
  • Open brackets must be closed in the correct order.
  • Every close bracket has a corresponding open bracket of the same type.


Concept:

Ok, so we have some parentheses which open and close frequently, maybe we have two open parentheses then one closing parentheses and we have to Determine if each of these open parentheses closed correctly, so absolutely we have stack


Solution:

  1. parenthes is open and we append it to the stack directly

  3. last condition: it's closing parentheses and it's type equalt to the last stack parenthes type so we pop

Equal Row and Column Pairs - leetcode #2352

Problem:

Given a 0-indexed n x n integer matrix grid, return the number of pairs (ri, cj) such that row ri and column cj are equal. A row and column pair is considered equal if they contain the same elements in the same order (i.e., an equal array).

Concept:

We have to loop through each column and row and compare between them, we should use hashtable to record the column result so we don't have to refind the column items each time we land on it through looping.


Solution:

first, we loop over each row and we loop over each item in this row, item index playing column index role, we check if column index is in the hashtable, if not we make a list of items in this column and save it in the hashtable. finaly we compare between row and column, if they are identical we increase the counter by 1, else counter stay on it's status and we go to the next column.
